# 🐱🤿 Scuba Cat Project (Kameralı El Hareketi ile Kontrol)


( arada gereksiz dosyalar olabilir sadece çalıştırmak için  python 3.13 sürümü indirin ve scuba_cat.py çalıştırın camera penceresi açılacaktır akımı yaptığınızda süpriz kediniz karşınızda :D )


Gerçek zamanlı kamera görüntüsünden el hareketlerini algılayarak ekranda interaktif bir “Scuba Cat” oluşturan Python tabanlı bir bilgisayarla görme projesi.

Bu proje, kullanıcıdan gelen el hareketlerini analiz eder ve belirli hareketlere göre ekranda bir  akım olan kedi videosu gösterir. Amaç; başta olmak üzere eğlence  , bilgisayarla görme, gesture recognition ve yaratıcı görsel geri bildirim sistemlerini bir araya getirmektir.

## 🚀 Özellikler
- Gerçek zamanlı kamera kullanımı
- El hareketi algılama
- Gesture'a göre kedi gösterimi
- Düşük gecikme

## 🛠️ Teknolojiler
- Python

## 📦 Kurulum
pip install python 

## ▶️ Çalıştırma
python scuba_cat.py

## 📁 Yapı
- scuba_cat.py
- scuba.mp4
